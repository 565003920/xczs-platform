## 项目简介

学程智枢是一款面向高校教师的教学过程智能诊断平台。通过对教务数据、课堂行为、学生评教、学业成果和教学资源**五源数据的融合分析**，构建从**数据采集 → 学情画像 → 模式诊断 → 对比分析 → 模式迁移 → 智能备课 → 课后反思 → 数据资产化**的完整闭环。

内置 **12 种教学模式模板**与**课程知识图谱**，支持**用户认证与数据隔离**，帮助不同教师独立管理自己的教学数据，从经验驱动转向循证教学决策。

### 数据闭环

```
采集 ──→ 融合 ──→ 画像 ──→ 诊断 ──→ 对比 ──→ 推荐 ──→ 应用 ──→ 反馈 ──→ 资产化
 ✅       ✅       ✅       ✅        ✅       ✅       ✅       ✅        ✅
```

### 核心功能

| 功能 | 说明 |
|------|------|
| 🔐 **用户认证** | JWT 登录，教师与管理员双角色，数据按用户隔离 |
| 📥 **数据导入** | CSV 批量导入课程、班级、学生、成绩、课堂观察、教学评价 |
| 📊 **教师工作台** | 点击课程卡片展开该课所有授课班级列表 |
| 🎯 **学情画像** | 六维雷达图 + 薄弱/优势知识点 + 成绩分布 |
| 🔍 **模式指纹** | 五维教学模式雷达图 + 匹配内置模板 + 改进建议 |
| 📈 **对比分析** | A/B 双雷达图 + Cohen's d 效应量 + 效能趋势 + 跨班排名 |
| 📚 **教学模式库** | 12 种内置模板 + 创建/编辑/删除 + 详情弹窗 |
| 🗺️ **知识图谱** | 力导向知识点依赖图 + 掌握率可视化 + 瓶颈预警 |
| 👤 **个体画像** | 学生个体知识热力图 + 趋势 + 薄弱/优势 |
| 🔄 **模式迁移** | 跨课程相似度 + 优质模式推荐（含模板完整信息） |
| 📝 **智能备课** | 从模板库加载环节 + 等比缩放时长 + 重难点标注 |
| 💡 **课后反思** | 达成度评估 + 偏差分析 + 改进建议 |
| 📦 **数据资产** | 资产目录 + 质量报告 + 数据血缘 + 效果仪表盘 + 审计日志 |
| 🔔 **通知中心** | 诊断完成/模式验证/质量预警通知 |
| 📋 **诊断报告** | 综合诊断 + 模式适配建议 + 可打印 |

## 快速开始

### 环境要求

- Python 3.10+ / Node.js 18+ / npm 9+

### 1. 克隆

```bash
git clone https://github.com/565003920/xczs-platform.git && cd xczs-platform
```

### 2. 后端

```bash
cd backend && pip install -r requirements.txt && python seed_data.py
uvicorn app.main:app --reload --port 8000    # → http://localhost:8000/docs
```

### 3. 前端

```bash
cd frontend && npm install && npm run dev     # → http://localhost:5173
```

## 推荐演示路径

1. 以 `zhang/123456` 登录 → 教师工作台看到 2 门课程
2. 点击「数据结构」卡片 → Drawer 显示 3 个授课班级，点击任一 → 学情画像
3. A/B 对比 → 选 1 班 vs 2 班 → 双雷达图 + Cohen's d
4. 模式指纹 → 查看 PBL 模板匹配结果
5. 模式迁移 → 选薄弱班(4 班) → 查看优秀班级推荐
6. 智能备课 → 选班级 + 输入"二叉树遍历" → 5 步教案
7. 课后反思 → 生成达成度 + 偏差分析
8. 退出登录 → 以 `admin/admin` 登录 → 多出「数据资产」菜单
9. 效果仪表盘 → 全景看板 + 热门模式 TOP5
10. 审计日志 → 查看完整操作记录
